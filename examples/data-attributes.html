üìö Data Attribute Reference

Complete list of available configuration attributes.

Attribute Type Default Description
data-swr - - Main identifier - required on container
data-swr-item - - Marks individual slide items
data-swr-aspect-ratio string '9:16' Aspect ratio (e.g., '9:16', '16:9', '1:1')
data-swr-loop boolean false Enable infinite looping
data-swr-autoplay boolean false Enable autoplay
data-swr-autoplay-interval number 5000 Autoplay interval in milliseconds
data-swr-keyboard boolean true Enable keyboard navigation
data-swr-touch boolean true Enable touch/swipe gestures
data-swr-transition number 350 Transition duration in milliseconds
data-swr-swipe-threshold number 50 Minimum swipe distance in pixels
